'Wizard Depths - Escape the Golem'
Playable Level



Developed using Unreal Engine 4, it features a controllable character with icicle projectile ability. The player must traverse the level to find their way out after being chased by a golem. It consists of multiple jumping sequences, puzzles, environmental animations and cutscenes.

The puzzles consist of finding hidden items to unlock a way out, cooling down a lava pump to stop powering the lock mechanism. as well as the possibility of creating shortcuts or assisting yourself with the icicles. It's designed with the 'quick escape' mentality in mind through which a simple linear path is hinted at and followed. The player has multiple interactable objects and UI text hints to help solve the puzzles.

Video or Image Projection
and CCTV Effects



This project was used to create multiple effects ready for use by a designer. These are:

  • Video projector with audio support
  • Realtime CCTV projector
  • Still image projector
  • Decal projector
  • Mirror object that uses camera to show reflection

The first 3 can have their shapes, brightness and other settings configured. The projectors are designed to allow easy implementation into a level to increase player immersion. Therefore settings like projectable distance, brightness and light shape among others are configurable.
